
Installatie handleiding
------------------------


Om van sledgehammer gebruik te maken importeer je sledgehammer_core naar de “$project/sledgehammer/core” map.
`git clone git://github.com/bfanger/sledgehammer_core.git sledgehammer/core`
Of voeg deze toe als gitmodule:
`git submodule add git://github.com/bfanger/sledgehammer_core.git sledgehammer/core`

SledgeHammer verwacht dat er een $project/tmp/ map bestaat.


Vanuit je bootstrap.php voeg je de volgende code toe:

define('ENVIRONMENT', 'development'); // andere optie is 'production'
$_SERVER['SERVER_ADMIN'] = 'email@waar.de.error.vestuurd.worden.nl'; // Deze wordt alleen in production modus gebruikt
include("sledgehammer/core/init_framework.php");
$AutoLoader->standalone = false; // Deze regel alleen toevoegen als je al gebruik maakt van een andere autoloader


Om te testen of alles werkt kun je de handige dump functie uitproberen: 
dump($var);

Daarnaast worden errors, warnings en notices nu door de SledgeHammer ErrorHandler afgehandeld. 

API Documentatie kan worden gegenereerd met de introspectie tool "DevUtils" zie: https://github.com/bfanger/devutils
